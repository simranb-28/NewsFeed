# 📰 NewsFeed Web Application

A React-based single page application that displays top global headlines using **NewsAPI**.
The app features **infinite scrolling, dark mode, category filtering, and optimized API calls** for a smooth browsing experience.

## 🚀 Features

* Infinite scrolling for seamless news browsing
* Category and country-based news filtering
* Real-time news search
* Dark mode with localStorage persistence
* Skeleton loaders for improved UX
* Responsive design for mobile and desktop

## 🛠 Tech Stack

* **React (Vite)**
* **JavaScript**
* **Axios**
* **React Context API**
* **CSS**

## ⚙️ Setup

1. Clone the repository

```
git clone https://github.com/simranb-28/newsfeed.git
cd newsfeed
```

2. Install dependencies

```
npm install
```

3. Create `.env` file

```
VITE_NEWS_API_KEY=your_api_key_here
```

4. Run the project

```
npm run dev
```
